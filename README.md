# Hello World 프로젝트

이 프로젝트는 간단한 웹 기술을 사용하여 만든 인터랙티브한 "Hello World" 페이지입니다.

## 주요 기능

*   **동적 텍스트 효과**: 마우스를 올리거나 상호작용할 때 텍스트가 생동감 있게 움직이는 애니메이션 효과를 포함하고 있습니다.
*   **오디오 재생**: 페이지와 함께 배경 사운드가 재생됩니다.
*   **실시간 시계**: 화면 우측 하단에 현재 날짜와 시간이 실시간으로 표시됩니다.

## 실행 방법

`index.html` 파일을 웹 브라우저로 열면 바로 확인하실 수 있습니다.
